Step 1: Quick Setup (2 minutes)

Get up and running in just a few commands:


# Clone the repository
git clone https://github.com/ciarareeve/STRXplorer
cd str-gwas-explorer

# Install Python dependencies
pip install flask pandas numpy plotly requests

# Run the application
python STRXplorer.py
That's it! The application will start at http://localhost:5000 with full functionality.

Step 3: Using the Platform

Once running, you can immediately:

  1. Browse Traits: View all available traits at /browse_traits
  2. Browse STR Loci: Explore STR locations at /browse_loci
  3. Generate Manhattan Plots: Click any trait to see all associated plots
  4. Check Status: Monitor system health at /database_status

Step 4: System Requirements

Minimal requirements for running the platform:

  • Python 3.8 or higher
  • 2GB RAM (4GB recommended)
  • 1GB free disk space
  • Modern web browser (Chrome, Firefox, Safari, Edge)

Required Python packages:


flask          # Web framework
pandas         # Data manipulation
numpy          # Numerical computing
plotly         # Interactive plotting
requests       # HTTP requests (for future data updates)
